# Promise Project Front-End

## Set up
1) clone
```bash
	$ git clone https://github.com/Promise-tech/promise-frontend.git
```
2) change directory
```bash
	$ cd promise-frontend
```
3) Install dependency
```bash
	$ npm install
```
4) Run the application
```bash
	$ np start
```
